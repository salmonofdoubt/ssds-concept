# Stability-Aware SDSS Concept Demo v1.0.2

## Overview

This repository contains a public concept demonstrator for a **stability-aware spatial decision support system (SDSS)** for prioritising **nature-based solutions (NbS)** in catchment and landscape planning.

The central idea is that prioritisation should not stop at producing a ranked map. It should also show **how stable or fragile those rankings are under reasonable changes in assumptions, data inputs, parameter weights, structural choices, and spatial scale**.

In practical terms, the demonstrator explores a decision architecture that is:

- transparent rather than black-box
- rule-explicit rather than impressionistic
- stability-aware rather than single-run deterministic
- governance-oriented rather than purely technical
- intended to support reproducible environmental decision making

## What this demo is

This is a **public research and concept demo**.

It is intended to:

- communicate the core logic of the proposed approach

## Project focus

The concept addresses decision support for NbS prioritisation with a specific emphasis on:

- water quality delivery
- biodiversity and habitat connectivity
- flood attenuation as a secondary objective
- uncertainty-aware ranking stability
- transparent portfolio logic
- governance-grade reproducibility
